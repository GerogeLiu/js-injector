What Can You Do?

Using this extension you will be able to inject javascript codes into webpages, in order to customize your most visiting websites, to have a more confortable web browsing.

In order to do so, create new Inject from create menu. You have to provide "URL", "Describtion" and "Javascript Code" that you want to inject. After creating an Inject, then Your Javascript Codes will run right before page is loaded.
